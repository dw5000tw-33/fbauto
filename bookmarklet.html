一鍵面板書籤（33 愛分享）

把下面這藍色面板按鈕直接拖到瀏覽器的書籤列。以後在任何頁面點書籤，就會開啟面板。

🧹開啟面板（33 版）
拖曳到書籤列 → 成功後再用右鍵可進行編輯（名稱建議：🧹 或 33掃地）。
第一次若被瀏覽器阻擋彈出視窗，請按允許。

超快兩步安裝

  1. 顯示書籤列：Ctrl+Shift+B（Chrome / Edge / Brave）。
  2. 把上方「開啟面板(33版)」按鈕拖到書籤列即可。

使用

開啟任一網站：臉書→ 點書籤「開啟面板」 → 面板視窗會跳出並完成注入腳本。

  • 此版本使用 Blob 注入（相容 Facebook CSP）。
  • 已加入 nonce 安全握手（只能由bookmarklet觸發）。

常見問題 FAQ

    Q1：點書籤沒反應？
    ➜ 檢查是否「阻擋彈出式視窗」，允許後再點一次。
    Q2：為什麼 Facebook 可以跑，其他網站卻不行？
    ➜ 有些網站設有嚴格的 CSP 會阻止注入；可先重新整理(F5)再試，或換在支援的網站使用。
    Q3：手機上能用嗎？
    ➜ 多數行動瀏覽器不支援 javascript: 書籤；請改用桌機。FB App 內建瀏覽器也不支援。
    Q4：按了之後頁面暫停？
    ➜ 正常現象，代表 DevTools 已打開；按 F8 或 DevTools 的「▶」繼續。
     Q5：要怎麼更新？
    ➜ 書籤不用重加；會抓最新面板版本。異常，可清快取或告知管理者。
    Q6：安全嗎？
    ➜ 書籤不含程式；就算有人直接開檔，也需通過檢核碼才會運作。
